# O que está pronto e o que ainda falta

## Pronto no pacote

- Portal G.One com visual original.
- AVA Moodle dentro da suite.
- ERP Globaltec dentro da suite.
- Docker Compose único.
- Gateway Caddy com HTTPS automático.
- Roteamento por subdomínios.
- Scripts de instalação, status, logs e backup.
- Botões no G.One para abrir AVA e ERP.

## Ainda depende da implantação

- VPS com Docker.
- DNS apontado para a VPS.
- Senhas fortes no `.env`.
- Primeiro acesso/configuração do Moodle.
- Conferência do usuário inicial do ERP.

## Não incluído nesta versão

- Login único/SSO entre G.One, AVA e ERP.
- Migração automática de alunos do G.One para Moodle.
- Sincronização automática curso Moodle ↔ projeto ERP.
- Pagamento online.

Esses itens são recomendados para a versão 2.
