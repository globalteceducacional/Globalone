# Mapa de módulos

- `apps/gone/` — G.One Portal; serviço `gone`.
- `apps/ava/` — Moodle/AVA; serviços `ava-db` e `ava-moodle`.
- `apps/erp/` — ERP; serviços `erp-db`, `erp-backend`, `erp-frontend`.
- `apps/teca-api/` e `apps/teca-web/` — TECA online; serviços `teca-db`, `teca-api`, `teca-web`.
- `apps/teca-flutter-source/` — código Flutter original enviado.
- `gateway/` — Caddy com HTTPS automático.
